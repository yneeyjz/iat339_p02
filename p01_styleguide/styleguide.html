Numbered Lists

CSS selector:

.numbered_list

Rendered element

  1. item 1
  2. item 2
  3. item 3

Sample codes:

<ol>
<li>item 1</li>
<li>item 2</li>
<li>item 3</li>
</ol>
CSS:
.numbered_list{
list-style-type: none; }
.numbered_list li{
counter-increment: item; }
.numbered_list li::before{
content: counter(item) ". "; color: #a3cf83;/*green*/ padding-right: 0.5rem; }
